# Scientific Skin Tech - Shopify Theme

## Estrutura do Tema

Este tema Shopify foi desenvolvido especificamente para a Scientific Skin Tech, incorporando a identidade visual e os valores da marca.

### Como Instalar

1. Faça login no Shopify Admin
2. Vá em Online Store > Themes
3. Clique em "Add theme" > "Upload zip file"
4. Faça upload do tema compactado

### Seções Incluídas

- **hero-section.liquid**: Hero principal com CTA
- **product-features.liquid**: Features da tecnologia BioSkin®
- **product-grid.liquid**: Grid de produtos
- **testimonials.liquid**: Depoimentos de clientes

### Personalização

As cores da marca podem ser ajustadas em `config/settings_schema.json`:
- Vermelho principal: #c4170c
- Vermelho escuro: #8a0f08

### Suporte

Para dúvidas sobre customização, consulte a documentação do Shopify Liquid:
https://shopify.dev/api/liquid
