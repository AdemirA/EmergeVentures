# Petvi - Shopify Theme

## Estrutura do Tema

Tema Shopify otimizado para e-commerce pet, desenvolvido para a Petvi com foco em conversão e experiência do usuário.

### Como Instalar

1. Acesse seu painel Shopify
2. Navegue até Online Store > Themes
3. Clique em "Add theme" > "Upload zip file"
4. Selecione o arquivo do tema

### Seções Personalizadas

- **hero-pets.liquid**: Hero com foco em pets e proprietários
- **product-benefits.liquid**: Benefícios dos produtos
- **testimonials-pets.liquid**: Depoimentos com pets
- **trust-badges.liquid**: Selos de confiança e certificações

### Cores da Marca

- Azul principal: #0066cc
- Azul escuro: #004080
- Laranja accent: #ff8f3d

### Funcionalidades

- Carrossel de produtos
- Filtros por tipo de pet
- Quiz para recomendação de produtos
- Sistema de assinatura

### Suporte

Documentação Shopify: https://shopify.dev/themes
