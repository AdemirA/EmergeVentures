# Vhita - Shopify Theme

## Estrutura do Tema

Tema premium para e-commerce de suplementos, desenvolvido para a Vhita com foco em confiança, qualidade e conversão.

### Instalação

1. Login no Shopify Admin
2. Online Store > Themes > Add theme
3. Upload zip file do tema
4. Publish quando estiver pronto

### Seções Incluídas

- **hero-wellness.liquid**: Hero focado em bem-estar
- **product-science.liquid**: Produtos com base científica
- **certifications.liquid**: Certificações internacionais
- **benefits-grid.liquid**: Grid de benefícios
- **subscription-cta.liquid**: CTA para assinatura

### Identidade Visual
